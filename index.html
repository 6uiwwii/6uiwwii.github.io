Temperature Measurement Humidity and Display

About Project

โปรเจคนี้เป็นการนำ Arduino UNO R4 WiFi มาเชื่อมต่อกับ Sensor DHT22 AM2032 เพือรับค่าอุณหภูมิและความชื้น จากนั้นนำไปแสดงบนเว็บไซต์ TEMPUI ที่มีฟังก์ชันการแสดงผลข้อมูลอุณหภูมิและความชื้นในหน้า Dashboard และมีการพยากรณ์อากาศใน 7 ถัดไป รวมถึงฟังก์ชันการแจ้งเตือนถึงอุณหภูมิที่เปลี่ยนแปลงไปแบบ Real-time
